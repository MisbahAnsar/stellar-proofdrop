## URL pattern

Replace `{hash}` with any transaction hash from the app (create, proof submit, approve, reject):

```
https://stellar.expert/explorer/testnet/tx/{hash}
```

Contract page:

```
https://stellar.expert/explorer/testnet/contract/CAPVLSAV2KGCGYXGJOGRR5XXBL6BDOV5WOOM64NLNDHUNKHWUIBQEBLC
```

## In the UI

After any on-chain action, the app shows:

1. A **View on Stellar Expert** button
2. The **full explorer URL** (clickable, opens in a new tab)

Task pages also list creation, proof, and review transaction links when available.
